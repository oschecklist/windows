The order of instructions / files / whatever I do when setting up Windows.

  1. Install updates.
  2. Set up a non-admin user to be used primarily and for all further steps.
  3. Uninstall default crap from start menu.
  4. Remove tiles and resize start menu.
  5. Make all icons in tray show up.
  6. Turn off Sticky Keys shortcut! (Press shift five times rapidly to trigger pop-up.)
  7. Install Google Chrome.
  8. Install Dropbox.
  9. Log into Dropbox / Chrome / GitHub.
  10. Set up File Explorer options from Control Panel (see this).
  11. Install scoop.
  12. Run
    Set-MpPreference -ExclusionPath "$(Resolve-Path '~/scoop')"
    Set-MpPreference -ExclusionPath "C:\ProgramData\scoop"
    to stop Microsoft Defender from bothering scoop apps. Note: The first line only works on an administrator account..so you have to
  13. Run scoop+apps.ps1 script.
  14. Set MPC-HC as the default media player.
  15. Install Steam.
  16. Set up git:
    git config --global user.email "paul.liverman.iii@gmail.com"
    git config --global user.name "Paul Liverman III"
    git config --global push.default simple
    ssh-keygen -t rsa -b 4096 -C "paul.liverman.iii@gmail.com"
    # ssh-agent -s # shouldn't be needed, should already be started
    ssh-add ~/.ssh/id_rsa
    cat ~/.ssh/id_rsa.pub
  17. Install Discord.
  18. Install OBS Studio.
  19. Install LAME for Audacity & FFMPEG for Audacity.
  20. Install Sony Vegas Pro 12 (it's on the network drive).
  21. Install Keybase.
  22. Use UTC time on the hardware clock: regedit script
  23. Run Update-Help in Powershell.
  24. Install these fonts.
  25. Install LibreOffice.
  26. Install MEGAsync.
  27. Install Backblaze.
  28. Copy startup program shortcuts to startup folder (here). (These programs won't run by default otherwise when the user is not an administrator.)
  29. Install TinyWall.
  30. Import saved TinyWall settings.
  31. Install CCleaner.
  32. Copy start menu program shortcuts to start menu folder (here). (These shortcuts open as administrator for their programs to run correctly.)
  33. Run CCleaner and tell it to save settings to INI file (under Options -> Advanced).
  34. Close CCleaner and copy saved CCleaner settings.
  35. Further customize settings.
