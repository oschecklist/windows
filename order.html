The order of instructions / files / whatever I do when setting up Windows.

  1. Install Google Chrome.
  2. Install Dropbox.
  3. Install scoop.
  4. Run Set-MpPreference -ExclusionPath "$(Resolve-Path '~/scoop')" Set-MpPreference -ExclusionPath "C:\ProgramData\scoop" to stop Microsoft Defender from bothering scoop apps.
  5. Run scoop+apps.ps1 script.
  6. Set MPC-HC as the default media player.
  7. Install Steam.
  8. Set up git: git config --global user.email "paul.liverman.iii@gmail.com" git config --global user.name "Paul Liverman III" git config --global push.default simple ssh-keygen -t rsa -b 4096 -C "paul.liverman.iii@gmail.com" # ssh-agent -s # shouldn't be needed, should already be started ssh-add ~/.ssh/id_rsa cat ~/.ssh/id_rsa.pub
  9. Install Keybase.
  10. Use UTC time on the hardware clock: regedit script
  11. Run Update-Help in Powershell.
  12. Install these fonts.
  1. Install Cosmoteer.
  2. Install bower for minetest mods: npm install -g bower
