The order of instructions / files / whatever I do when setting up Windows.

  1. Install Google Chrome.
  2. Install Dropbox.
  3. Log into Dropbox / Chrome / GitHub.
  4. Set up File Explorer options from Control Panel (see this).
  5. Install scoop.
  6. Run
    Set-MpPreference -ExclusionPath "$(Resolve-Path '~/scoop')"
    Set-MpPreference -ExclusionPath "C:\ProgramData\scoop"
    to stop Microsoft Defender from bothering scoop apps.
  7. Run scoop+apps.ps1 script.
  8. Set MPC-HC as the default media player.
  9. Install Steam.
  10. Set up git:
    git config --global user.email "paul.liverman.iii@gmail.com"
    git config --global user.name "Paul Liverman III"
    git config --global push.default simple
    ssh-keygen -t rsa -b 4096 -C "paul.liverman.iii@gmail.com"
    # ssh-agent -s # shouldn't be needed, should already be started
    ssh-add ~/.ssh/id_rsa
    cat ~/.ssh/id_rsa.pub
  11. Install Discord.
  12. Install OBS Studio.
  13. Install LAME for Audacity & FFMPEG for Audacity.
  14. Install Keybase.
  15. Use UTC time on the hardware clock: regedit script
  16. Run Update-Help in Powershell.
  17. Install these fonts.
  18. Install LibreOffice.
  19. Install MEGAsync.
  20. Install Backblaze.
  21. Copy startup program shortcuts to startup folder (here).
  22. Install TinyWall.
  23. Import saved TinyWall settings.
  24. Install CCleaner.
  1. Install Cosmoteer.
  2. Install bower for minetest mods: npm install -g bower
